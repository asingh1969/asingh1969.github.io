Evernote

Evernote is a cloud service that offers both free and premium packages for uploading and storing notes and documents and accessing said documents from any Evernote supported device. Currently Evernote has versions for most big name devices such as Windows and Mac desktops and personal computers and iOS, Android, and Windows Phone.

Storage

To hold all the user data Evernote uses something called Shards. A shard is a pair of Supermicro boxes that run Debian and are managed by MySQL(Engberg2). Each of the Supermicro boxes contains two virtual machines. The primary virtual machine holds all the user data and Evernote’s core application stack while the secondary is a copy of the primary that is kept synchronously replicated. There is also a “UserStore” account database, which also run and maintained MySQL that keeps track of some information about each account.

Searching

To allow searching through your documents Evernote has 28 servers the process images. The Evernote development team made an “Advanced Imaging and Recognition” software, AIR(Engberg2), to process the images.

Networking

Evernote uses Border Gateway Protocol to send traffic through their primary service provider, NTT, and secondary provider, Level 3. The traffic is then sent to Evernote’s A10 Load Balancers.(Engberg2)

Analytics and Reporting

Data warehousing for Evernote’s analytics and reports are done by a three tier system. Warehousing and primary data is held by a ten node Hadoop cluster, a three node ParAccel database is set for querying derived tables, and a JasperReports server generates reports(Engberg1).

Other

Evernote also has a suite of servers for all there other needs, such as their corporate website, blog, forums, etc and the Evernote website is built off of the Thrift API(Engberg2).
